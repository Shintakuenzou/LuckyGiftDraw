## Instalando raffle-gifts-vite

Para instalar o raffle-gifts-vite, siga estas etapas:

Linux e macOS e Windows:
```
git clone https://github.com/Shintakuenzou/LuckyGiftDraw.git
```
```
npm install
```
```
npm run dev
```
